# Customer Segmentation using KMeans

This project uses KMeans clustering to segment customers based on:
- Age
- Average Spend
- Visits per Week
- Promotion Interest

## Model
- Algorithm: KMeans
- Optimal Clusters: 4 (Elbow Method)

## How to Run

1. Install requirements:
  ``` pip install -r requirements.txt

2. Run Streamlit app:
  `` streamlit run app.py
